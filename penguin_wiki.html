企鹅（英语：Penguin），全称“企鹅（贵族朋友）”，是 Aurora Project 企划的主要技术支持者与网站开发者。因其在开发过程中频繁制造 Bug、并在正常网页中植入惊悚元素（Jump Scare），被企划组成员戏称为“甲级战犯”。

生平

企鹅的具体身世不详，据称来自南极洲。在加入 Aurora Project 之前，他曾是一名普通的 PHP 程序员，后转型为全栈（Bug）工程师。

争议事件

Jump Scare 事件

2026年1月，在 Aurora 官方网站上线前夕，企鹅在“项目加入”弹窗的“我不支持”按钮中，恶意植入了高频噪音与恐怖图片弹窗代码。此举导致多名早期测试用户受到惊吓，手机甚至产生剧烈震动。尽管本人辩称这是“为了增强用户互动的沉浸感”，但普遍被视为其“战犯”行为的确凿证据。

代码拖延

据内部知情人士（kawako）透露，企鹅经常以“在抓鱼”为由推迟功能上线。实际上，他多数时间都在研究如何让 CSS 写得更像 2000 年代的风格。
